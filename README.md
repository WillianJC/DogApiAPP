# 📱 Dog Api APP - Android

Una aplicación Android simple para consumir API's, desarrollada con **Kotlin** y **XML** en Android Studio.

## 🚀 Características

- ✅ Interfaz de usuario intuitiva y limpia
- ✅ Consumir API's
- ✅ Actualizar la API

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Kotlin
- **UI**: XML Layouts
- **IDE**: Android Studio
- **Plataforma**: Android
